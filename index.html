USTA Team Manager
One-time setup — connect your Google Sheet
STEP 1 — Create your Google Sheet

Go to sheets.google.com and create a new spreadsheet. Name it USTA Team Manager. Copy the Sheet ID from the URL:

https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID_HERE/edit
STEP 2 — Google Cloud project & API key

1. Go to console.cloud.google.com → New Project
2. Enable the Google Sheets API and Google Identity Services
3. Create an API Key (restrict to Sheets API)
4. Create an OAuth 2.0 Client ID (Web application type)
   — Authorized origins: your GitHub Pages URL (e.g. https://yourname.github.io)
   — Also add http://localhost for local testing

STEP 3 — Share your sheet

Share the Google Sheet with anyone with the link can view. Players will write via OAuth (their own Google login), and the captain writes via OAuth too.

STEP 4 — Set captain password
STEP 5 — Team name

Settings saved to your browser. Players visiting the URL will be prompted to sign in with Google.
